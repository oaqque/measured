The plan now treats morning mobility as a daily default session from `2026-04-21` through the Sydney Marathon on `2026-08-30`. Rather than keeping mobility as a scattered supporting note on only a few dates, the training source now carries one explicit mobility note for every day in that span.

For dates that did not already have a mobility note, new `Morning Mobility` note files were added. For the existing `Mobility Only` days, those notes were retained and expanded so they now contain the same full lower-body routine instead of a short placeholder.

Each daily note now embeds the full `Ideal Daily Version` from `metaanalysis/MORNING_MOBILITY.md`, with the recovery-day adjustment called out directly in the note. The downstream effect is that the mobility habit is now explicit in the authored source all the way through the marathon block instead of being implied by one metaanalysis document and a few isolated note files.
