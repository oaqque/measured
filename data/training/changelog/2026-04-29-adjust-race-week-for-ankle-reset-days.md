The current week from `2026-04-27` to `2026-05-03` has been rewritten around the stated right-ankle peroneal tendon irritation and the emerging pattern of taking reset days between easy runs. The original race-week sharpening plan no longer matched reality: Monday's easy run was completed, no run or gym showed up on Tuesday, and there is still no provider evidence for any sharper taper work on Wednesday morning.

Rather than pretending the old `8 km` sharp session, steady Thursday, and strides still make sense, the week now alternates easy running with mobility-only reset days. Tuesday and Thursday become no-run mobility days, Wednesday is reduced to `5 km` easy, Friday stays a short `4 km` ankle-check run, basketball is explicitly cut, and the weekly volume line now reads as `9 + race` instead of a clean `50 km` taper week.

Sunday's HOKA Runaway half marathon is still on the calendar, but it is no longer written as a hard-target race effort. The note now makes it explicit that this should be treated as a controlled decision-day effort and that warm-up symptoms can override the original race intent. The downstream impact is a week that protects the ankle first and treats racing as conditional on actually arriving there settled enough to run.
