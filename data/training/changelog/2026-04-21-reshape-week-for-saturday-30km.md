The active week from `2026-04-20` to `2026-04-26` has been reshaped around a planned `30 km` long run on Saturday at roughly `6:00 /km`. That makes Saturday a real key aerobic session instead of a filler day, so the rest of the week can no longer pretend Thursday and Sunday should stay as originally written.

Thursday's `10 km` steady run has been reduced to an easy `6 km` aerobic support run and the lighter support gym session has been cut back to mobility only. Friday stays as a short `6 km` recovery run, but the note now makes it explicit that the day should stay cheap and uncomplicated rather than drifting upward in pace or volume.

Saturday's basketball-only placeholder has been replaced with the `30 km` long run and explicit fueling practice. Sunday's planned `14 km` long run has been removed and replaced with full rest so the week does not stack two long aerobic loads back to back. Because Thursday is no longer a steady-run progression, the authored note-graph link for that session progression has also been removed so the training graph matches the revised plan instead of pointing at a workout that no longer exists.

The downstream impact is a week that now concentrates its real cost into Tuesday plus Saturday and protects the days around that choice well enough to keep the change coherent.
