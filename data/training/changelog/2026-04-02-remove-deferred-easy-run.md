The planned `8 km` easy aerobic run for Thursday was removed after the Wednesday threshold session was completed and recorded as planned.

That deferred easy run only made sense while the week still carried uncertainty about where the threshold session would land. Once the Wednesday session was real and logged, the extra Thursday run stopped reading as a clear part of the week and instead duplicated the recovery intent already covered by the surrounding schedule.

Thursday now keeps only the lighter support gym session, and the weekly running total drops accordingly.
